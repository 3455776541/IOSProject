# IOSProject
IOSProject项目是一个以MVC模式搭建的开源功能集合，基于Objective-C上面进行编写，意在解决新项目对于常见功能模块的重复开发，IOSProject对于项目的搭建也进行很明确的划分，各个模块职责也比较明确，IOSProject也引入的一些常用第三方插件、宏定义、工具帮助类等;
